Cost Function

Each particle in a PSO search evaluates a model configuration using a cost function. The cost function is the summation of the weighted population cost and the weighted performance cost. The weights provide some tuning, the designer can use the weights to determine the importance of each cost in the search.

The summation and weights work as follows:

Population cost

Population cost is the weighted sum of the population size of each component in that model configuration.

Performance cost

Performance cost is dependent on the selection of performance measure. In either case the PSO is looking to reduce the cost as much as possible. Target is the threshold or target performance the designer is looking for. Measured is performance measure of the evaluated model configuration.
