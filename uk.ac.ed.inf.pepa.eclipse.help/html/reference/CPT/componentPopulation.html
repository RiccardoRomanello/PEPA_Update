Component population

The number of components in a system. If we have:
Component[x]
x then is the population size of the resource in the system.

Component population range

The designer must specify an upper and lower limit to each component population size. This provides a component population range.

Model configuration search space

The set of all possible model configurations.

Model configuration

A model configuration is any model who has each of its component population sizes inside of their user defined component population ranges.
