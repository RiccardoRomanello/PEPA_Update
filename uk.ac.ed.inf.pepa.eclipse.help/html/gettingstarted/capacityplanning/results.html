The above is a screen shot of the PEPA Eclipse plug-in project. On the top right we see the views for the results.

Overview

The CPT offers three different types of result

  1. The Capacity Planning List/Table View
  2. The Capacity Planning Tree View
  3. The csv format file

It is possible to click and drag, or maximise the right hand View pane. In here we find the two Views. The remaining result, the .csv file, is found in the location set in the Capacity Planning Wizard. The .csv contains all model configurations seen by the search, and also the configuration information used to perform the search.
