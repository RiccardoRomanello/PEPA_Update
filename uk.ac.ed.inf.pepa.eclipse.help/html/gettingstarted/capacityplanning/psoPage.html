Starting a PSO: Particle Swarm Optimisation configuration

The above is a screen shot of the first page of the CPT PSO wizard. In this page you are configuring the PSO parameters:

  1. Experiments - Choose the number of times you would like to run a PSO.
  2. Generations - Choose the maximum number of 'moves' each particle can do.
  3. Population - Choose the number of particle agents searching the space.
  4. Set the Original velocity weight*.
  5. Set the Personal best weight*.
  6. Set the Global best weight*.

*See "Particle veloctiy" in the CPT reference for more details...
