The Table view


The above is a screen shot of the CPT Table or List View in the PEPA Eclipse plug-in project.

Overview

This view provides a report of all model configurations found during the search. It is ranked by cost, with the least cost model configuration found at the top of the View. It is designed to provide the designer with a quick overview of the returned configurations. For more detail of each model configuration please see the Tree View.

  1. Cost. The cost of the model configuration as determined by the cost function
  2. Population. The populations of each component type in the model configurations.
  3. Performance measure. The measured performance of the model configuration.
  4. Total population. The summed total of all component populations in the model configuration.
