The Tree view


The above is a screen shot of the CPT Tree View in the PEPA Eclipse plug-in project.

Overview

This view complements the List View, and provides more detail about the search. It provides a summary of the results at the top of the Tree, and the individual details of each model configuration.

1. Results Evaluation details

  1. Results evaluation: MET TARGET PERFORMANCE. This means the search returned at least one model configuration which meets the user defined performance target.
  2. Number of model configurations which met the target
  3. Percentage of models that converged. This gives some indication of how many of the ODE evaluations converged, if an ODE evaluation did not converge the fitness is set to 10000, as the cost function has to assume the evaluation failed.
  4. Total results. Total number of model configurations found.
  5. Average model population. The average summed total population.

2. Model configuration details

  1. ModelConfigurationCandidate-X MET TARGET PERFORMANCE. Indicates if this model configuration has managed to meet the target.
  2. Component population.
  3. Total cost. The value of the model configuration's cost as determined by the cost function. The output of the cost function.
  4. Measured performance. The measured performance of the model configuration.
  5. Performance target. The performance the user defined.
  6. Percentage of targets met. If the user had set more than one performance target this reports what percentage of them met their target.
  7. Performance cost. This model configuration's performance cost in terms of the cost function.
  8. Population cost. This model configuration's population cost in terms of the cost function.
  9. Total population.
  10. PSO parameters. The parameters used to find this model configuration. This is useful if the search was a driven search, the user can use these values to perform a single PSO search.
  11. Converged. Did the ODE evaluation converge.
  12. Run time. How long did this model configuration take to evaluate.
